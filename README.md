#Col App Project Template

My personal grunt-init app template

##Dependencies

`sudo npm install -g grunt-init`

##Installation

`git clone git@github.com:colingourlay/grunt-init-col-app.git ~/.grunt-init/col-app`

##Usage

```
mkdir app-dir
cd app-dir
grunt-init col-app
npm install

```
